# Web 开发路径

## 路径说明

这条路径适合想从零开始学习前端开发的同学。建议按照 HTML、CSS、JavaScript、Vue、接口联调与部署的顺序推进，每一段都配合在线编辑器做即时练习。

## 建议阶段

### 1. HTML 与 CSS 基础

- 学会页面结构、常见标签和语义化写法
- 掌握盒模型、布局、间距、响应式设计

### 2. JavaScript 核心语法

- 变量、函数、对象、数组
- 条件判断、循环、作用域、异步基础

### 3. Vue 组件开发

- 组件拆分
- Props / Emits
- 状态管理和页面组合

### 4. 接口联调与部署

- 使用 `fetch` 或 `axios`
- 处理加载状态和报错
- 把项目部署到线上环境

## 配套建议

学完每个阶段后，都可以去 `/playground` 写一个小例子验证理解，再进入下一阶段。
