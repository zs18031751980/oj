# 数据科学路径

## 路径说明

这条路径适合希望把 Python 用在数据清洗、分析和可视化上的同学。重点不是只学语法，而是建立完整的数据处理思路。

## 建议阶段

### 1. Python 编程基础

- 变量、函数、列表、字典
- 文件读写和基础脚本组织

### 2. pandas 与 numpy

- 表格数据读取
- 缺失值处理
- 常用统计和向量化操作

### 3. 数据可视化

- 使用 matplotlib / seaborn 做图
- 学会用图表达趋势、分布和对比

### 4. 模型与实验记录

- 了解基础机器学习流程
- 学会记录实验参数和结果

## 配套建议

可以把清洗逻辑先在本地项目中完成，再将关键函数放到在线编辑器中验证边界输入。
