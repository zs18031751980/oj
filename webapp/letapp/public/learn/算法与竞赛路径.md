# 算法与竞赛路径

## 路径说明

这条路径适合 OJ 刷题、面试准备和竞赛训练。建议把“会写代码”和“会分析复杂度”同时作为目标。

## 建议阶段

### 1. 数据结构基础

- 数组、链表、栈、队列、哈希表
- 树、堆、图的基础认识

### 2. 搜索与排序

- 二分
- DFS / BFS
- 常见排序算法及其复杂度

### 3. 动态规划专题

- 状态定义
- 转移方程
- 滚动优化

### 4. 题解复盘与优化

- 记录错题
- 复盘边界条件
- 比较不同写法的时间和空间开销

## 配套建议

在编辑器里多输入自定义测试数据，专门验证边界条件，例如空输入、极值和重复元素。
